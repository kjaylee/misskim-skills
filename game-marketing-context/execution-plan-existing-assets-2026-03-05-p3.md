# 기존 자산 마케팅 실행안 (P3, 48시간)

- 작성일: 2026-03-05 (KST)
- 목표: 신규 게임/툴 제작 없이 기존 게임 자산으로 48시간 내 신규 플레이어 유입 증대
- 사용 자산: 기존 텔레그램 게임 딥링크, 기존 플레이 영상/스크린샷, 기존 채널(텔레그램/쇼츠)

## 1) 유입 채널 2개

### 채널 A — 텔레그램 오운드 채널
- 집행 방식
  - 고정 메시지를 "이번 주 추천 게임 1개" 단일 CTA로 교체
  - 24시간 내 포스트 2회 발행(도전형 카피 / 보상형 카피)
- 자산 활용
  - 기존 하이스코어 캡처 이미지
  - 기존 딥링크 + UTM(`source=tg_owned`)

### 채널 B — YouTube Shorts/TikTok
- 집행 방식
  - 기존 영상 클립 3개를 15~20초로 재편집 후 48시간 내 업로드
  - 설명란 CTA를 텔레그램 딥링크 1개로 통일
- 자산 활용
  - 기존 플레이 하이라이트 영상
  - 기존 썸네일/자막 템플릿 + UTM(`source=shorts_rented`)

## 2) 실험 가설 3개

1. **CTA 가설**
   - 가설: "지금 플레이"보다 "친구 점수 이겨보기" 카피가 CTR을 높인다.
   - 성공 기준: 경쟁형 카피 CTR이 기준 카피 대비 +20% 이상

2. **훅 가설**
   - 가설: 영상 첫 2초에 도전 메시지(예: "10초 안에 가능?")를 넣으면 완시율이 오른다.
   - 성공 기준: 도전 훅 버전의 3초 유지율 +15%, 완시율 +10%

3. **링크 구조 가설**
   - 가설: 포스트 내 다중 링크보다 단일 링크가 게임 시작 전환율을 높인다.
   - 성공 기준: 단일 링크 버전의 시작률 +15%

## 3) KPI

### 핵심 KPI (48시간)
- 채널별 Link CTR
- 딥링크 클릭 대비 게임 시작률(Start Rate)
- 콘텐츠 1개당 신규 플레이어 수

### 보조 KPI
- 실험 유입 코호트 D1 리텐션
- 플레이어 공유 클릭률
- Shorts 유입자의 텔레그램 구독 전환율

### 목표값
- 텔레그램 CTR ≥ 4.5%
- Shorts/TikTok CTR ≥ 1.8%
- 딥링크→시작률 ≥ 55%
- 신규 플레이어 수 기준선 대비 +25%

## 4) 48시간 실행 체크리스트

### 0~6시간 (세팅)
- [ ] 기존 게임 3개 중 실험 대상 1개 확정
- [ ] 영상/이미지 자산 6개 선별
- [ ] UTM 파라미터 규칙 확정 (`source`, `campaign`, `creative`)
- [ ] KPI 추적 시트 생성

### 6~24시간 (1차 집행)
- [ ] 텔레그램 포스트 1개 발행 (CTA A)
- [ ] 쇼츠/틱톡 2개 업로드 (훅 A/B)
- [ ] 고정 메시지 단일 CTA로 교체
- [ ] 6시간 단위 중간 성과 체크

### 24~36시간 (최적화)
- [ ] 하위 성과 크리에이티브 30% 교체
- [ ] 텔레그램 포스트 2개차 발행 (CTA B)
- [ ] 쇼츠/틱톡 1개 추가 업로드 (승리 훅 복제)
- [ ] 댓글/DM 반복 질문 3개를 카피에 반영

### 36~48시간 (마감)
- [ ] KPI 최종 집계(CTR/시작률/신규 유입)
- [ ] 가설 3개 판정(Win/Lose/Iterate)
- [ ] 다음 7일 확장안 1개 결정(승리 조합만 확장)
- [ ] 5줄 결과 요약 기록
