# 하네스 생성기 최소안

작성 시각: 2026-04-02 10:31 KST

## 목표
미스 김이 새 작업을 시작할 때 바로 사용할 수 있는 내부용 하네스 생성기 최소안을 만든다.

## 이번 구현 범위
- 작업 입력을 받는다
- 스펙 문서 초안을 만든다
- 스폰 지시문 초안을 만든다
- 상태 파일 초안을 만든다
- 테스트와 완료 조건을 강제로 포함시킨다

## 출력물
- `specs/<job-id>/plan.md`
- `specs/<job-id>/spawn.md`
- `.state/pipelines/<job-id>.json`

## 성공 기준
- 명령 한 번으로 위 세 파일이 생성된다
- 테스트, 완료 조건, 산출물 경로가 비어 있지 않게 강제된다
- 예시 한 번 실행으로 실제 파일 생성 검증

## 레드팀
- 공격 하나: 너무 큰 생성기를 만들면 또 하나의 유지 대상이 된다
- 공격 둘: 템플릿이 약하면 이름만 하네스고 내용은 빈 껍데기일 수 있다
- 방어: 이번에는 최소 생성기만 만들고, 필수 항목 강제에 집중한다
- 합의: 위험 수용
