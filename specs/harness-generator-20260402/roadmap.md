# 하네스 생성기 성장 로드맵

## phase 0 완료
- 최소 생성기
- 계획 문서 생성
- 스폰 지시문 생성
- 상태 파일 생성

## phase 1 완료
- 프리셋 지원
- 기본 산출물 자동 생성
- 테스트 추가

## phase 2 완료
- 실제 스폰용 문서 직출력
- 레드팀 문단 자동 삽입
- 실행 원칙 자동 삽입
- 스폰 레디 파일 생성

## phase 3 완료
- 하네스 상태 파일과 동급 권한 위임 봇 연결
- 대기 상태 기준 자동 판정
- 틱 처리기 추가
- 상태 파일에 마지막 위임 기록 저장

## phase 4 예정
- 리서치, 구현, 리팩터, 배포 프리셋 고도화
- 검증 형식 자동 추가
- 실제 서브에이전트 호출 문자열까지 생성

## phase 5 예정
- 미스 김 작업 시작 훅과 연결
- 오 분 무응답 규칙 반영
- 파이프라인별 후속 행동 자동 추천
