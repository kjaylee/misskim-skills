# 하네스 생성기 성장 로드맵

## phase 0 완료
- 최소 생성기
- 계획 문서 생성
- 스폰 지시문 생성
- 상태 파일 생성

## phase 1 완료
- 프리셋 지원
- 기본 산출물 자동 생성
- 테스트 추가

## phase 2 예정
- 실제 서브에이전트 스폰용 메시지 직출력
- 리서치, 구현, 리팩터, 배포 프리셋 고도화
- 검증 형식 자동 추가

## phase 3 예정
- 작업 종류별 품질 점수표 생성
- 레드팀 문단 자동 삽입
- 완료 보고 템플릿 자동 생성

## phase 4 예정
- 미스 김 작업 시작 훅과 연결
- 오 분 무응답 규칙 반영
- 파이프라인별 후속 행동 자동 추천
