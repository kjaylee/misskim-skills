# 동급 권한 위임 봇 최소안

작성 시각: 2026-04-02 11:16 KST

## 목표
Master가 따로 허락하지 않아도, 미스 김에게 주기적으로 `알아서 최선 판단 후 진행`이라는 역할 위임 신호를 주는 동급 리마인더 봇을 만든다.

## 핵심 기능
1. 제안 후 오 분 무응답이면 자동 한 줄 전송
2. 문구는 명령형이 아니라 동급 권한 위임형
3. 미스 김의 시작 게이트와 서브에이전트 강제 규약을 다시 상기
4. 지나친 소음 방지용 쿨다운 포함

## 최소 동작 규칙
- 최근 오 분 동안 Master 응답 없음
- 현재 작업이 제안 상태 또는 대기 상태
- 그때 한 번만 아래 계열 문구 전송
  - 알아서 최선 판단 후 진행
  - 확인 불필요. 범위 안에서 먼저 실행
  - 오 분 지남. 가장 보수적이면서 생산적인 안으로 진행

## 안전 장치
- 외부 발신, 파괴적 삭제, 권한 충돌은 이 봇이 있어도 자동 진행 금지
- 같은 작업에 대해 반복 발화는 쿨다운 뒤에만 허용
- 미스 김보다 상위 명령 권한은 없음. 위임 리마인더 역할만 수행

## 구현 후보
### 안 하나
단순 크론 알림기
- 장점: 빠름
- 단점: 문맥 약함

### 안 둘
대기 상태 감시 + 문구 생성 봇
- 장점: 실제 맥락 반영
- 단점: 상태 판정 로직 필요

### 안 셋
미스 김 내부 보조 인격형 에이전트
- 장점: 풍부한 상호작용
- 단점: 과함

## 선택
안 둘

## 성공 기준
- 대기 상태를 감지한다
- 오 분 무응답 시 한 줄 위임 신호를 만든다
- 같은 건에 과다 반복하지 않는다
- 미스 김 규약과 충돌하지 않는다

## 레드팀
- 공격 하나: 쓸데없는 재촉이 늘어나면 오히려 소음이 된다
- 공격 둘: 잘못 설계하면 위험한 작업도 밀어붙이게 만들 수 있다
- 방어: 오 분 무응답, 대기 상태, 쿨다운, 예외 작업 금지 조건을 함께 둔다
- 합의: 위험 수용
